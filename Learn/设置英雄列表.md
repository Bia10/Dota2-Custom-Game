# 设置英雄列表



如何让游戏地图只能够选择指定的英雄？

在 `<MAP>/scripts/npc` 目录下，新建 `herolist.txt` 文件，内容如下

File: `<MAP>/scripts/npc/herolist.txt`

```
//	可以选择的英雄
//
// <key>	<value>
// Hero		currently on/off, will be # of instances (-1 = infinite)
// 			值为0时，表示禁用，其他值表示启用
//

"CustomHeroList"
{
	"npc_dota_hero_axe"		"-1"	// 斧王
}
```

上面的代码只定义了 **斧王** ，所以可以选择的英雄也只有 **斧王**。
