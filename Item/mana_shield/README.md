## 魔法护盾



![](src/icon/mjz_mana_shield.png)

名称：魔法护盾

效果：

​	创造一个魔法护盾，消耗魔法值以吸收的伤害。

​	伤害吸收：60%

​	每魔法吸收伤害：2.5

​	*如果在死亡時魔法护盾是激活的，那么复活時也是激活的*

价格：8400

合成：林肯法球（5050） + 闪避护符（1400） + 圆盾（200） + 精气之球（1200） + 抗魔斗篷（550）

加成：力量 +10、敏捷 +10、智力 +10、攻击力 +10

​	+5 生命恢复速率

​	+5 魔法恢复速率

​	+160 生命值

​	+160 魔法值
