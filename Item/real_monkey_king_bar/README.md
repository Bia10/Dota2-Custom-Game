## 真正的金箍棒



![](src/icon/real_monkey_king_bar.png)

名称：真正的金箍棒

效果：

- 主动：棒击大地

  - 让金箍棒变大，并将其挥起猛击地面，眩晕一条直线上的敌人 2 秒并造成 300%  致命一击伤害。具有克敌机先效果。吸血：25%。距离：1200。

- 棒击大地 Debuff：减速

  - 移动速度减缓 30%, 攻击速度降低 30%, 持续时间 2 秒。

  

加成：攻击速度 +20、攻击力 +88

合成：金箍棒（4175） + 金箍棒（4175）

价格：8350

冷却时间：30

耗蓝：15
