# 狂战斧 系列



![](src/icon/mjz_battlefury.png)

名称：狂战斧 系列

效果：

​	主动：**砍伐**

​		砍伐一根指定树木，或者摧毁一个守卫。

​	被动：**压制**

​		对非英雄单位进行物理攻击会造成额外伤害，近战英雄为24点，远程英雄为7点。

 	被动：**分裂**

​		普通攻击时对目标周围最远650距离的锥形区域内敌人造成60%物理伤害。（仅近战有效）
