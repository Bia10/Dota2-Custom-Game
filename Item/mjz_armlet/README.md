## 臂章 系列



![](src/icon/mjz_armlet.png)

名称：臂章 系列

效果：

​	激活：**邪恶之力**

​		  激活后邪恶之力将提升31点攻击力、25点力量和4点护甲，但每秒失去54点生命值。

开启邪恶之力后生命流失并不会导致死亡，关闭邪恶之力后力量降低也不会导致死亡。
