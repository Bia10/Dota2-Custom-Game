# Boots Clock Up



![](src/icon/mjz_boots_clock_up.png)

名称：Boots Clock Up

效果：

 	主动：**Clock Up**

​	提升移动速度并使你可以穿越单位，转身也更快。近战英雄提升122%移动速度，远程英雄提升110%移动速度，状态持续15秒。

​	**移动速度加成(近战)**：122%

​	**移动速度加成(远程)**：110%

​	**转身速率(近战)**：1

​	**持续时间**：15

​	**冷却时间**：30

加成：护甲 +10、攻击力（近战） +20、攻击力（远程） +15 、移动速度 +18%

合成：相位鞋 (1480) + 风灵之纹(250) + 卷轴 (670)

价格：1480 + 250 + 670 = 2400
