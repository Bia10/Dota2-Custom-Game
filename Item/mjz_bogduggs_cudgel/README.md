# 波格达格的巨棍



![](src/icon/mjz_bogduggs_cudgel.png)

名称：波格达格的巨棍

效果：

 	被动：**粉碎**

​	仅近战 - 所有攻击速度减半，但是攻击在目标 295 范围内造成溅射伤害，并且眩晕 0.4 秒。

​	**粉碎范围**：295

​	**自身攻击速度降低**：40%

​	**自身基础攻击间隔降低**：40%

​	**眩晕时间**：0.4



等级：25

加成：力量 + (5 + 0.3 * 宝物等级)、攻击力 + (10 + 0.4 * 宝物等级) 

合成：碎颅锤 (2950) + 食人魔之斧 (1000) + 卷轴 (950)

价格：2950+ 1000 + 950 = 4900
